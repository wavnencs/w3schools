Tags

  • Only the content inside the body section is displayed in a browser.
  • HTML headings are defined with the h1 - h6 tags.
  • Attributes provide additional information about an element.
  • Attributes usually come in name/value pairs like: name="value".
  • In HTML, a color can be specified by using an RGB, or a HEX value.
  • Without a forward slash, you might generate two requests to the server.
  • Inline - by using the style attribute in HTML elements.
  • External - by using an external CSS file.
  • _self - Opens the document in the same window/tab, as default.
  • _parent - Opens the document in the parent frame.
  • _blank - Opens the document in a new window or tab.
  • _top - Opens the document in the full body of the window.
  • The head element contains meta information about the document.
  • The h1 defines the most important heading.
  • The strong element defines bold text, with added semantic importance.
  • The em element defines italic text, with added semantic importance.
  • The del element defines deleted (removed) text.
  • The ins element defines inserted (added) text.
  • The sub element defines subscripted text.
  • The sup element defines superscripted text.
  • The blockquote element defines a section from another source.
  • The pre element defines preformatted text.
  • The address element defines contact information.
  • The abbr element defines an abbreviation or an acronym.
  • The samp element defines sample output from a computer program.
  • The kbd element defines keyboard input.
